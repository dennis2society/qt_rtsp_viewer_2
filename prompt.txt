This following link points to a working Qt6 C++ application (CMake) to display and record (using ffmpeg) a selectable  RTSP video stream from a selectable camera (using  persistently saved and removable RTSP URLs). It also allows to enable multiple image processing methods via a sidebar to enable things such as simple blur, grayscale conversion, brightness/contrast/color-temperature, motion tracking and motion vectors, an overlay motion graph, FPS/resolution overlay and face detection (using OpenCV). Video playback and processing is done in a QThread.
